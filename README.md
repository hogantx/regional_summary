#  regional summary

This code creates 16 wmf images for the regional summary analysis.
The 16 files are saved to the directory with names denoting region and segment.
e.g., apac_aib.wmf

The week_number is determined based on the sysdate.
This weeknumber determines
when to terminate POS
when to terminate EoH
woi calculation

The plot dimension is set to 10 x 5.
This allows linking to powerpoint.

Two files are required:
regional_summary_desktop.csv
regional_summary_aib.csv
